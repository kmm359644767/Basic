0.5-1万/月

职位信息

1、参与智慧教室后台系统的需求调研和需求分析，撰写相关技术文档；

2、负责系统架构和功能开发以及前后端技术交互的接口设计；

3、负责与智慧教室平台子系统、子设备接口对接开发；

4、系统开发测试、部署和集成；

5、对新技术有专研精神，善于技术攻关。



岗位要求：

1、计算机相关专业本科及以上学历，需要有一定的系统分析能力，1年以上开发经验；

2、 能熟练使用Python，掌握Python下的调试以及数据结构设计，熟悉Python与Java、C语言相互调用；

3、 熟悉 MySQL，MongoDB，Redis 中的一个或者多个；

4、 熟悉 Django 框架，能独立开发 Web 服务和 RESTful API 服务，有独立完成的线上作品优先。

5、 有 Docker 使用经验优先。

6、 有 RabbitMQ 或其他消息队列使用经验优先。

7、 具备良好的沟通能力和较强的团队合作精神。

职能类别： 互联网软件开发工程师 软件工程师

关键字： Python 开发工程师 软件工程师

公司信息

        广东新源信息技术有限公司成立于2006年，注册资金2010万，总部位于广州，在湖南、广西、云南、贵州、北京等地设立有分公司。 公司专注于教育、政府及大型企业等行业信息化建设，为客户提供量身定制的解决方案。公司业务范围涉及产品总代理、系统集成、建筑智能化、软硬件开发等多个方面，是一家专业的集产品销售、技术服务、工程实施、产品研发等为一体的综合服务商。
        公司自成立以来坚持走持续创新的道路，通过持续的研发投入和对行业背景的深入研究，为广大用户提供基于应用场景的创新解决方案。公司自主研发的数字化校园平台、学生全生命周期管理系统、智慧教室解决方案、智慧校园节能监管系统、智慧实训解决方案、基于人脸识别技术的智慧通行方案、智慧校园解决方案等，深受广大客户的好评。并与国内知名高校开展产学研合作，勇于挑战新一代信息技术最前沿课题，进行技术攻关与科技成果转化，用高科技为行业导航。公司自主研发的专利产品近20项，目前已经成为物联网及IPV6领域技术研发的领导品牌。
        公司具有计算机信息系统集成三级资质，安防壹级资质，并通过了ISO9001、ISO14001、ISO18001、ISO20000、ISO270001等体系认证，连续多年被广东省工商局评为“守合同重信用企业”。公司被认定为“广东省高新技术企业”，“广州市研发建设机构”， “广州市科技小巨人”、广东省优秀信用企业等。
       公司坚持 “诚信、专业、一切以客户为中心”的宗旨，为广大客户提供专业、优质、贴心的服务。凭借专业的技术实力，行业口碑深受众多客户的认可，与众多行业客户建立了良好的合作关系，树立了众多典型案例。
