职位信息

职责描述：

1、负责公司内部项目服务端的研发工作； 

2、 负责公司基础平台的研发工作； 

3、负责编写设计开发文档、设计手册、应用手册相关技术文档；

4、 负责数据的获取及分析 。


任职要求： 

1、3年以上的Python相关软件开发经验；

2、 计算机基础理论扎实，熟悉数据结构、算法、操作系统、TCP、HTTP 等计算机基础知识； 

3、扎实的编程语言python/GO功底； 深入了解 Django、Flask、Tornado 等主流 Python Web 框架； 

4、 熟悉 MySQL  Redis 等数据库 并有良好的数据库设计和优化能力； 

5、具备较强的编程能力，了解正确的编码规范；

6、 熟悉 UnixLinux 操作系统；

7、 具有较强团队合作、发现问题、解决技术问题的能力，思路清晰，可提出创新解决方案； 

8、 熟悉自动化运维以及 Ansible zabbix ELK  Docker 等相关工具者优先；

9、热爱软件研发工作，有强烈的创新精神并能付诸实施，能积极学习业界新技术。

公司信息

      广州南天电脑系统有限公司(Guangzhou NANTIAN Computer System Co.,Ltd) （以下简称“广州南天”）是云南南天电子信息产业股份有限公司在华南地区的专业控股子公司，成立于1992年。目前员工人数超过600人。
    云南南天电子信息产业股份有限公司(以下简称南天信息或南天)，是以软件业务、产品服务业务、集成服务业务、智慧城市业务、创新业务等五大板块业务为主体，以服务为发展方向的现代化高科技企业，具有三十多年建设金融行业和国家部分重点行业信息化工程的丰富经验。1999年8月18日，南天信息在深圳证券交易所挂牌上市，股票代码000948。
       南天信息分设昆明总部和北京总部，在北京、上海、广州、西安、武汉、昆明等地均设有全资或控股子公司，132个维修站遍布全国。以“与世界信息潮流同步，成为著名信息化服务企业”为企业愿景，以“金融信息化行业领先、信息技术产业一流”为发展定位，南天建立起了覆盖全国的应用开发及销售服务体系，可为客户提供高质量的IT专业服务。
       在软件业务方面，南天是国内银行业十大解决方案主要提供商之一，拥有多个世界领先的应用软件产品及解决方案，实施的金融应用软件产品解决方案和应用系统超过1000余个，应用的网点达5万个，产品深受国内银行认同。公司可为客户量身定制成熟的应用系统与解决方案，提供业务咨询服务软件产品销售、软件开发与运维、人力资源外包等各种专业化服务。此外，南天软件产品还广泛覆盖政府、电信、资产管理公司、财务公司等多个行业。
       在集成服务业务方面，南天成功打造了数据中心（DC）、智能数据中心管理（IC）和主媒体呼叫中心（CC）三大业务方向，与IBM、思科、HP、AVAYA、BMC、华为、浪潮、微软等国际知名IT厂商建立有长期而良好的战略合作伙伴关系。目前，公司已成为国内数据中心服务的主要提供商，业务已广泛渗透到国内各大商业银行和政府等行业的数据中心建设和服务领域。
       在产品服务业务方面，南天是国内专业生产金融电子化专用设备的主导厂商，目前产品已全面覆盖国内各大商业银行、专业银行及新生银行，并出口到二十多个国家和地区。
       在智慧城市业务方面，南天凭借在信息化领域多年积累的系统经验，以自有核心技术和关键应用为基础，快速建立智慧城市业务，立足云南，辐射全国。
广州南天还是南天IBM主机业务责任主体，是IBM国内指定的的银行主机核心应用研发中心（暨演示中心、培训中心、技术支持中心）(简称zCITA中心)，承担南天基于IBM大型主机环境的软件开发实施服务，面向国内的中小银行用户提供基于IBM主机平台的解决方案。
       广州南天作为国家信息技术标准（ITSS）工作组运维专业组副组长单位，参与了国家信息技术系列标准制定工作，把公司多年来在信息技术服务方面积累的知识与经验转换为标准，为参与IT服务的各个环节提供行动指南，推动信息技术服务产业链的健康、有序发展。作为第一起草单位主持编制了《信息技术服务 运行维护 第4部分：数据中心规范》，并参与了多项标准的起草编制、评审、修改等工作。
       此外，在“十三五”期间，南天还将大力发展工业云、智能制造、云计算、大数据、互联网金融等创新业务，积极响应国家关于大力推进工业4.0进程的产业政策。
       为适应IT行业更新迅速、竞争激烈的特点，多年来，南天始终坚持致力于提升专业实力，打造企业品牌。
